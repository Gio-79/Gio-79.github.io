Arduino Servo Controller

Windows



With the Arduino Servo Controller you can control up to 5 servos with your computer
(the software is only compatible with Windows).

Connecting

To use the servo controller, load the script on the arduino and go to: Tools>Ports>COM3,4,5,6...(Arduino...). Enter the used port (where it says COM4 by default). in the other input fields, enter the bound rate of the port and then click on connect. If a green ball appears, it means the port is connected, if it is red, it means the connection failed.

Controlling

In the software you will see 5 sliding modules, each with an input field on the right and left. In the field on the left, enter the smallest angle that the servo should accept and in the field on the right, the largest value. When you have done this, slide the blue sliders to the desired angle, lastly to send the instructions to the arduino press upload

With the upload function, the software sends the angle value to the Arduino individually with an interval of 1500ms. This is saved by the Arduino in a (int => integer) variable. When the upload process is complete, the angle value is accepted for each connected servo.
